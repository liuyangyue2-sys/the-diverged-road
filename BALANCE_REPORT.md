# 平衡机制说明 v3

本版把结局判定改为“候选结局打分制”：

1. 归零结局 Z01-Z04 和违法被抓 H05 仍然最高优先。
2. 体制特殊线、创业保底线、隐藏线优先于普通结局。
3. 大厂、科研、家庭、创业、基层、AI 等路线型结局优先于 E01/E14/E15。
4. E01 门槛提高为四维高分，避免它抢走 E10/E11/E12/E13。
5. E14/E15 只作为泛化稳定/普通兜底，不再大量截胡。
6. 最后仍保留最低项/最高项兜底，确保一定能出结局。

理论可达结局包括：E01-E18、H01-H05、Z01-Z04。隐藏/归零结局仍然故意较难，正常局不会过量出现。

Supabase 已强制配置：所有用户默认写入 sessions_latest，不再依赖每台设备本地填写配置。

## 本地合成测试

已用合成 state 做一次触发验证：Z01/Z02/Z03/Z04、H01-H05、E01-E18 均能被命中。实际玩家分布仍会受题目路线和随机创业/违法概率影响，但不再出现泛结局大面积截胡路线结局的问题。
